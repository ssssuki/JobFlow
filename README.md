# JobFlow

## Track Your Job Search

Project in Action - [JobFlow](https://jobflow.onrender.com/)




#### Run The App Locally

```sh
npm run setup-production
```

- rename .env.temp to .env
- setup values for - MONGO_URL, JWT_SECRET, JWT_LIFETIME

```sh
node server
```

- visit url http://localhost:5006/
